# File-system-driver
Simple implementation of file system driver as a console utility with possibility to invoke Unix-like filesystem syscalls.

The list of implemented Unix-like system calls:
- mount
- umount
- pwd
- ls
- cd
- mkdir
- rmdir
- link
- unlink
- truncate
- rm
- create
- filestat
- open
- read
- write
- close
- symlink
