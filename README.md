# File-system-driver
Simple implementation of file system driver as a console utility with a set of commands (most are Unix-like).

Some of Unix-like implemented commands:
- mount
- umount
- pwd
- ls
- cd
- mkdir
- rmdir
- link
- unlink
- truncate
- rm

Other commands:
- create
- filestat
- open
- read
- write
- close
- symlink
